What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (such as an HTML or XML document) as a tree of objects, making it possible for programming languages like JavaScript to interact with the document.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you use DOM selection methods provided by the document object.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for its child elements. Instead of attaching event listeners to each child element individually, the event listener on the parent uses the event's target property to identify the child element that triggered the event.
